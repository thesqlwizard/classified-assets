# classified-assets

Public asset repository for CLASSIFIED Marketing tool.

## Folder Guide

### browse-nodes/
Add a folder named after each browse node (lowercase, hyphens for spaces).
Drop product images inside. The app will auto-pull them.

Example:
```
browse-nodes/mens-premium-grooming/product1.jpg
browse-nodes/stanley-cups/product1.jpg
```

### templates/collage/system/
Your curated collage front card templates. Name them template-01.jpg, template-02.jpg etc.

### templates/lifestyle/system/
Your curated lifestyle scene templates.

### templates/*/user-uploaded/
Auto-populated by the app when users upload their own templates.

### styles/moods/
Visual mood reference images: grainy.jpg, clean.jpg, moody.jpg, editorial.jpg, dreamy.jpg

### styles/luts/
Color grade / LUT reference images.

### branding/logos/
Auto-populated by the app when users upload brand logos.

### reference/
Auto-populated when users upload look/feel reference images.

### history/
Auto-populated with saved ad outputs per session.
